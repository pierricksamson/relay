Une ligne de code entre vos scripts et votre DM Discord

Vos scripts parlent.
Discord écoute.

Relay est un relais minimal : une clé API, un endpoint, un message privé qui arrive sur Discord. Pas de compte à créer, pas de serveur à configurer. Permet de recevoir facilement les alertes de vos propres applications, scripts ou codes directement sur Discord

01

Connexion Discord uniquement

Pas de mot de passe à gérer. Votre identité, c'est votre compte Discord — Relay récupère votre ID, votre pseudo et votre avatar, rien de plus.

02

Une clé, un endpoint

Générez une clé pk_... depuis le dashboard et envoyez un POST /api/send depuis n'importe quel langage.

03

Historique clair

Chaque notification envoyée, réussie ou échouée, reste consultable dans votre dashboard.

En pratique

import requests

requests.post(
    "{{ request.url_root.rstrip('/') }}/api/send",
    json={
        "api_key": "pk_votre_cle",
        "message": "Mon serveur est démarré ✅",
    },
)
